# Setup leafwax data management

Interactive setup wizard for configuring leafwax data management
options.

## Usage

``` r
setup_leafwax_data(reset = FALSE)
```

## Arguments

- reset:

  Logical, whether to reset to defaults

## Examples

``` r
if (FALSE) { # \dontrun{
# Run interactive setup
setup_leafwax_data()
} # }
```
