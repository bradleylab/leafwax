# Mini posterior draws

Compact subset of posterior draws used in tests and smoke checks. Not
for analytical use; see
[`load_posteriors`](https://bradleylab.github.io/leafwax/reference/load_posteriors.md)
for the full v10 posteriors shipped under `inst/extdata/posteriors/`.

## Usage

``` r
mini_posteriors
```

## Format

A list of draws indexed by model name.
